# RCME Static Website

Simple static RCME website scaffolded for GitHub Pages.

## Pages

- `index.html`
- `about.html`
- `program.html`
- `fee-structure.html`
- `infra.html`
- `accommodation.html`
- `faculty-domestic.html`
- `contact.html`
- `404.html`
- `CNAME`

All internal links are relative and the Apply Now link points to `https://apply.rcmb.in`.

## Deploy on GitHub Pages

1. Push the contents of this folder to a GitHub repository.
2. In repository settings, enable GitHub Pages.
3. Publish from the selected branch and root folder.
4. Custom domain is configured with `CNAME` for `rcme.co.in`.

## Domain DNS

Point `rcme.co.in` to GitHub Pages using the DNS records GitHub recommends for apex domains. Add `www` as a CNAME to `jaimandal10.github.io` if the `www` version is also needed.
